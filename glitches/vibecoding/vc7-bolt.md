Bolt.new: otevřeš prohlížeč, napíšeš zadání — třeba „Kalkulačka BMI s historií měření" — a za pár sekund máš funkční aplikaci. Žádná instalace, žádné nastavení.

Silné stránky: full-stack aplikace (vizuální část + databáze) a stažení celého kódu jako ZIP. Slabost: u velmi složitých projektů může ztrácet kontext. Tip: čím přesnější popis, tím lepší výsledek.

! Přesně! Bolt.new funguje kompletně v prohlížeči — žádná instalace ani nastavení. Otevřeš, napíšeš, stavíš.

+++
StackBlitz, firma za Boltem, původně postavil WebContainers — technologii, která umožňuje spouštět Node.js (serverový JavaScript) přímo v prohlížeči. To bylo technicky náročné a dříve nemožné. Bolt.new je přímá aplikace této technologie — proto jako první nástroj zvládl full-stack vývoj přímo v prohlížeči bez instalace. Je to fundamentální technický průlom, ne jen AI vrstvička nahoře.

Full-stack aplikace (mající frontend i backend) jsou výrazně cennější než čistě frontendové. Frontend je to, co vidíš — tlačítka, formuláře, barvy. Backend je to, co běží na serveru — ukládání dat, přihlašování, výpočty. Dřív vyžadoval backend server, databázi a deployment — věci, které jsou pro začátečníka náročné. Bolt tyto složitosti abstrahuje a generuje funkční backend automaticky.

Stahování kódu jako ZIP je klíčová funkce pro dlouhodobost projektu. Pokud Bolt jednou přestane existovat nebo zdraží, tvůj kód máš u sebe. Nejlíp je kód hned po vygenerování stáhnout na GitHub — bezplatnou platformu pro ukládání kódu. GitHub je „záloha a verze" projektu, portál, kde vývojáři sdílejí a spolupracují. I kdybys nikdy neprogramoval, GitHub účet pro tvoje vibecoded projekty je dobrý nápad.
